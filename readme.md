# Yoshimo's Remorse

## Overview

Was that truly the ending to Yoshimo's adventures with the Bhaalspawn? Is there simply no way to redeem himself or explain his actions to the PC?

Yoshimo's Remorse is a mod for BG2EE that makes Yoshimo a playable character in Throne of Bhaal. However, he is not what you expect him to be...
